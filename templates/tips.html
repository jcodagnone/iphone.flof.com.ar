Tips and Tricks

  • iphone.flof.com.ar is great to find places near a known location (for example Santa Fe y Callao or Corrientes 400). But what happens if you don't know where you are located?

    It's not an issue! Just launch the Maps Application use the GPS to locate the position or browse the map to the location and in the Search Box type http://i.flof.com.ar (don't forget to type http:// part). This will display all the places tagged in flof for the location. If you change the position of the map just set the focus to the Search Box, and hit the search button again.
